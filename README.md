# AI War Room

The AI War Room is an innovative platform designed to provide businesses with a strategic advantage by offering real-time, data-driven competitive insights. This project showcases a robust full-stack application leveraging modern web technologies and artificial intelligence to transform raw market data into actionable intelligence for informed decision-making.

## Key Features

### 1. Comprehensive Competitive Analysis

Gain a holistic view of the competitive landscape with in-depth comparisons across critical business metrics. The platform visualizes and analyzes:

*   **Revenue**: Track and compare revenue performance of competitors.
*   **Market Share**: Understand market penetration and dominance.
*   **Net Promoter Score (NPS)**: Gauge customer sentiment and loyalty.

This feature provides a quantitative foundation for strategic planning, allowing users to identify strengths, weaknesses, opportunities, and threats within their market segment.

### 2. Dynamic Competitor Intelligence Dashboard

The interactive dashboard serves as a central hub for deep company insights. It aggregates and presents vital information about competitors, enabling users to quickly grasp market trends, competitor strategies, and potential disruptions. The dashboard is designed for clarity and ease of use, making complex data accessible at a glance.

### 3. AI-Powered Strategic Chat

Leverage an intelligent AI chat interface to answer complex strategic questions. This feature integrates live data from diverse sources, including Reddit for real-time public sentiment and Yahoo Finance for financial performance, to provide nuanced and up-to-date responses. The AI chat acts as a strategic advisor, offering insights that go beyond traditional data visualization.

## Technical Architecture

The AI War Room is built with a modern, scalable, and performant technology stack:

*   **Next.js**: Utilized for its powerful React framework capabilities, enabling server-side rendering (SSR) and static site generation (SSG) for optimal performance, SEO, and developer experience. It forms the robust foundation of the user interface.
*   **Supabase**: Employed as a backend-as-a-service (BaaS) solution, providing a PostgreSQL database, authentication, and real-time subscriptions. Supabase simplifies backend development, allowing for rapid feature iteration and secure data management.
*   **Groq API**: Integrated for its high-performance language processing capabilities, powering the AI chat feature. Groq's efficient inference engine ensures rapid and accurate responses to complex queries, enhancing the user experience.
*   **Tailwind CSS**: Used for utility-first CSS styling, enabling rapid and consistent UI development. Tailwind CSS ensures a clean, responsive, and visually appealing design across the application.
