# AI War Room 

A competitive intelligence platform that battles any two companies head-to-head using live data.

## What it does
- Compares revenue, market share, and NPS across competitors
- Competitor Intel dashboard with deep company insights
- AI chat that answers strategic questions using live Reddit + Yahoo Finance data

## Built With
Next.js · Supabase · Groq API · Tailwind CSS

## Setup

1. Clone the repo
2. Install dependencies:
\```bash
npm install
\```
3. Create `.env.local` and add:
\```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
GROQ_API_KEY=your_key
\```
4. Run locally:
\```bash
npm run dev
\```
